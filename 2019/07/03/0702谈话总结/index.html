0702谈话总结

July 3, 2019 本文总阅读量次
  1. 学会合作，去积攒自己的实力，当你有好的主意，实力会成为你完成你理想的敲门砖，你的朋友也会成为你成功路上重要的伙伴，找到他们，记住他们，和他们合作。
  2. 未来不管方向如何，保持创新（积极的想法，考虑两面性的能力，远处的习以为常可以成为近处的创业之道）时刻告诉自己，哪里都可以有创造的火花，抓住并且让他不断燃烧。
  3. 保持联系，和朋友保持联系，了解内外的差异，更新彼此的信息。
  4. 在海浪翻滚的大海下，总有沉静安定的海体，要去从容应对上下翻滚的海浪，不忘坚持千米以下的广阔宁静（理解很多，目前的理解是在面对生活周遭，心情起伏，要从容地应对这些会影响你状态的事，但是与此同时，伟大的目标，平和的心性要一直坚持）
  5. 人际关系。关于人际，首先要做的是不去做无用功的交际，浪费时间去和无关紧要的人应酬交道。不如提升自己，让自己变得更强，去帮助那些真正关心你陪伴你的人。（舍己从人）
  6. 喜欢的生活和舒适圈的思考。毁掉一个人最好的方式就是让他过喜欢的生活？有疑问，如果单从走出舒适圈的话来说，我赞成，但喜欢的生活不等于不走出舒适圈。等看了具体的文章再说吧。

往期陈庭讲话

  1. 李政道的导师，每天只问他今天做了什么+明天要做什么！！！
  2. 想法多且要去尝试，不要害怕失败，这只是一种经历，只要有一个成功了，都值得骄傲。形成固定的积极的思维方式，不要让自己害怕失败！
  3. 交换思维。
  4. 看人，徳，能，学习能力！！！
